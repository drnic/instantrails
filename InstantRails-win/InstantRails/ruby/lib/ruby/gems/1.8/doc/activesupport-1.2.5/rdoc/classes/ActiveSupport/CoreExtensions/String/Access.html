Module ActiveSupport::CoreExtensions::String::Access
In: lib/active_support/core_ext/string/access.rb

Makes it easier to access parts of a string, such as specific characters and substrings.

Methods

at   first   from   last   to  

Public Instance methods

at(position)

Returns the character at the position treating the string as an array (where 0 is the first character).

Examples: 

  "hello".at(0)  # => "h"
  "hello".at(4)  # => "o"
  "hello".at(10) # => nil
first(limit = 1)

Returns the first character of the string or the first limit characters.

Examples: 

  "hello".first     # => "h"
  "hello".first(2)  # => "he"
  "hello".first(10) # => "hello"
from(position)

Returns the remaining of the string from the position treating the string as an array (where 0 is the first character).

Examples: 

  "hello".from(0)  # => "hello"
  "hello".from(2)  # => "llo"
  "hello".from(10) # => nil
last(limit = 1)

Returns the last character of the string or the last limit characters.

Examples: 

  "hello".last     # => "o"
  "hello".last(2)  # => "lo"
  "hello".last(10) # => "hello"
to(position)

Returns the beginning of the string up to the position treating the string as an array (where 0 is the first character).

Examples: 

  "hello".to(0)  # => "h"
  "hello".to(2)  # => "hel"
  "hello".to(10) # => "hello"
